# TpFinalNivel2_Callava

Aplicación de escritorio para la gestión de artículos utilizando base de datos SQL Server.

## Descripción

Este proyecto consiste en un **gestor de artículos** desarrollado como trabajo final de un curso de programación.
La aplicación permite administrar un catálogo de productos realizando operaciones de alta, baja, modificación y búsqueda,
utilizando una base de datos provista y una arquitectura en capas.

El sistema está pensado para **uso local**, como práctica de conceptos fundamentales de desarrollo de aplicaciones de escritorio.

---

## Tecnologías utilizadas

- Lenguaje: **C#**
- Framework: **.NET Framework**
- Tipo de aplicación: **WinForms**
- Base de datos: **SQL Server**
- Arquitectura: **Capas (Dominio, Negocio, Datos, Presentación)**

---

## Funcionalidades

- Alta de artículos
- Modificación de artículos
- Eliminación de artículos
- Listado de artículos en DataGridView
- Búsqueda simple
- Búsqueda avanzada con filtros
- Manejo de imágenes (agregar imágenes desde la PC)
- Validaciones de campos
